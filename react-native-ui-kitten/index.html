What is UI Kitten

UI Kitten is a React Native implementation of Eva Design System. It contains a set of general purpose UI components styled in a similar way. You focus on business logic and the Kitten takes care of visual appearance. And the most awesome thing: the themes can be changed in the runtime, without any need of reloading the application.

Components

Almost 20 general purpose components designed and tested to save development time while providing smooth app experience

Theming System

Use several themes and create your own ones. UI Kitten is based on Eva Design system and it will keep your app clean

Guides

Comprehensive clear documentation with the tons of examples can answer any questions that a developer might have
